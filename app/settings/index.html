Settings

Manage workspace trust, recovery, category framing, and browser-specific controls without breaking durable GTM truth.

Workspace-level
Backup and restore
Export your synced workspace as a JSON backup file, import a previous backup, or fully clear the workspace.
Export creates an offline recovery file on this device. Import overwrites the current workspace with the selected backup, then resyncs durable workspace truth. Delete clears the signed-in workspace and sends you back to onboarding.
Account-level
Role and Onboarding
Changed roles or starting fresh? Re-run the onboarding flow to reconfigure your workspace. Your existing data stays safe.
Role setup belongs to your signed-in account and activation path. Resetting onboarding should retune the workspace without deleting ICPs, deals, or signals.
Browser-only
Demo Mode
Preview the app with sample data to see what a fully configured workspace looks like. Demo data is isolated from your real workspace.
Demo mode is a local sample workspace. It should never overwrite your signed-in cloud workspace, and exiting demo should return you to real workspace truth.
Workspace-level
Product Category
Select which product category you're selling. This controls which discovery framework appears in your live call navigator.
Category is stored with the workspace because it affects downstream discovery, call guidance, and GTM framing across modules.
