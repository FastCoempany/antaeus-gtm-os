# Dashboard Soft-Cut Command Docket Spec

This file freezes the chosen dashboard-critical account card language.

Selected system:
- `System 01F`
- `Slice 01`
- strengthened mark prominence
- pushed-right value column

## Component role

This component is the canonical right-rail account unit for Dashboard.

It exists to say:
- this account matters now
- why it matters now
- where the pressure lives

without turning into:
- editorial copy
- decorative signal soup
- a fake heat dashboard

## Locked outer shell

The outer shell is fixed.

Locked:
- width
- height
- vertical mass
- radius
- left-edge accent
- split into two objects

The card may not grow taller or fatter because the interior gets redesigned.

## Locked macro composition

The card is two pieces:

### Left work block
- account name
- three signal rows
- one short footer line

### Right dock
- amount
- three compact row echoes

This split must remain legible.
The dead middle seam must not return.

## Locked signal structure

Each account uses three rows only:

1. `Risk`
2. `Proof`
3. `Motion`

Each row contains:
- one prominent mark
- one row label
- one short value
- one short clarifier

## Interior tuning now locked

### Marks
- larger than helper glyph size
- enough character to feel like true signs
- not tiny, apologetic icons

### Value column
- pushed further right
- allowed to occupy the seam created by the soft cut
- not crowded against the row label

### Clarifier line
- short
- concrete
- must not become paragraph copy

### Right dock lines
- not a meter
- not a scale
- row signatures only

## Meaning rules

The card must imply:
- urgency
- triage
- consequence
- operating seriousness

It must not imply:
- dashboard decoration
- analytics theater
- editorial essaying

## Forbidden moves

Forbidden:
- fake meters
- filler chips
- wrapping token rows
- new accent systems inside the card
- extra boxes inside the left work block
- a second competing sign system inside the dock
- widening or heightening the card shell

This spec should drive:
- the canonical dashboard mockup
