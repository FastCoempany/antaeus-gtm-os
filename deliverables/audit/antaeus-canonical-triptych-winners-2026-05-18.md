## Audit interpretation rule

When a Program 6 refacing audit lands on a room, the **canonical surface to compare the ship against** is:

1. If a `-selected-` companion file exists (Deal Workspace, Future Autopsy main + stage-strip), use it directly.
2. If a refinement file exists (Dashboard's `softcut-canonical`, Territory Architect's `signal-field-refinement`, Discovery's `ledger-spine-canonical`), use the refinement — it supersedes the original triptych panel.
3. If only the triptych exists, use the specific named panel from the founder's message (e.g. "Variant 02 / Talk Loom" for Cold Call Studio).
4. For greenfield rebuilds (Onboarding, Negotiation, Founding GTM, Readiness, Settings), no triptych comparison applies — audit against the canon §4 room description + the rubric + the persona walks.
