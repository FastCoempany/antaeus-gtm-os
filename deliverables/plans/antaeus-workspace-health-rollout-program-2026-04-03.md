# Antaeus Workspace Health Rollout Program
Date: 2026-04-03
Status: Active
Owner: Codex / Antaeus post-restructure product wave

## Purpose
Carry the `Workspace health` thesis out of Dashboard and into selected deep rooms without changing:

- GTM frameworks
- module substance
- methodology
- strategic seriousness
- preserved room brains
- shell hierarchy

This is a room-intelligence wave, not a shell wave and not a methodology rewrite.

## Why now
The command surface is already:

- ranked
- explainable
- fed by room truth

So the next leverage point is making deep rooms show:

- what is compounding
- what is still weak
- why the room matters now

without forcing the user to go back to Dashboard for every health read.

## Product thesis
Each serious room should be able to say:

- what is alive here
- what is underbuilt here
- what the next integrity risk is
- what is compounding because of the work already done

The user should feel:

- `The room knows what is getting stronger and what is still fake.`

## Non-goals
This program does not:

- redesign room methodologies
- replace room primary workflows
- alter command routing
- add new rooms
- turn rooms into analytics dashboards
- duplicate the full Dashboard graph inside rooms

## Program shape

### Phase 1
Deep-room health surfaces, first slice

Deliver:

- `Deal Workspace` health surface
- `Signal Console` health surface
- shared room-health styling
- health logic sourced from real room state

### Phase 2
Room-health expansion

Deliver:

- `PoC Framework`
- `Discovery Studio`
- possibly `Advisor Deploy`

### Phase 3
Cross-room health coherence

Deliver:

- shared terminology
- compounding vocabulary alignment
- weak-link consistency

### Phase 4
Acceptance and tuning

Deliver:

- burden review
- continuity review
- no-break review

## Definition of done

- selected rooms show local health truth clearly
- rooms expose weak links without becoming report surfaces
- compounding is visible inside the room where work happens
- no module brain is rewritten
- no connective tissue breaks
