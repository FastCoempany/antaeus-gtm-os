# Antaeus Workspace Health Rollout Phase 2 Implementation Spec

Date: 2026-04-03
Owner: Codex
Status: Executed / recorded

## Scope

Phase 2 formalizes the second rollout slice defined in:

- `deliverables/plans/antaeus-workspace-health-rollout-program-2026-04-03.md`

This slice extends Workspace Health from:

- `Deal Workspace`
- `Signal Console`

into:

- `PoC Framework`
- `Discovery Studio`

## Objective

Make room health visible where proof and live-call compounding actually happen without:

- turning the room into a report
- repeating room-bridge copy
- diluting methodology
- adding decorative dashboard mimicry

## PoC Framework surface

Show:

- proof posture
- live deal linkage
- scope boundary count
- readout ownership status
- `Compounding`
- `Still weak`

Use real room truth from:

- linked deal presence and stage
- quality score and band
- success criteria count
- boundary count
- readout owner
- timing pressure

## Discovery Studio surface

Show:

- call posture
- advance rate
- worked-move loop strength
- deal linkage
- `Compounding`
- `Still weak`

Use real room truth from:

- logged-call count
- advance rate
- worked-move count
- linked deal presence
- active framework
- agenda quality and handoff state

## UI rules

- place the section in the first fold
- keep the section compact
- use the shared `.workspace-health*` system
- cap `Compounding` and `Still weak` to the top 2 items each
- keep the section actionable and diagnostic, not educational

## Acceptance

- `PoC Framework` health reads like proof integrity, not a report
- `Discovery Studio` health reads like live-call compounding, not a report
- both sections sit near the top without displacing the room’s main work
- no module brain or room workflow is altered
- shared terminology stays aligned with the rollout program

## Follow-on

After Phase 2 acceptance, the next controlled expansion targets are:

- `Advisor Deploy`
- `Call Planner`

These should continue the same pattern:

- local truth
- compact posture metrics
- top 2 `Compounding`
- top 2 `Still weak`
