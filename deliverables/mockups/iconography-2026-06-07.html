The set is ours.

A borrowed icon library tells the operator this is the same kind of tool as everything else — before they read a word. So the set is drawn, not borrowed: a heavier 2px keyline, flat terminals and sharp joins that read as technical drawing rather than friendly UI, resting on the same edge-rule the cards use, accented once and only where the meaning is. Navy nearly everywhere; color is rationed, not a palette.

01Construction & signaturespec §1 · what makes it proprietary

Every glyph is drawn on a 24px grid with a 2px keyline. The one choice that stops it looking generic: flat (butt) terminals and mitered joins, never the round caps of the default libraries. Two signature elements tie the set to the four-system language — the edge-rule (a baseline the glyph rests on, echoing the anchored edge and the gauge) and the rationed tick (a single accent stroke at the icon's active point, in the role color, rationed the way orange is).

02The complete setspec §3 · noun · verb · system · status

Every glyph stands for a sacred noun, a verb the operator acts with, or a system or status meaning — nothing decorates, nothing fills space. Signal carries the blue intelligence tick; the dominant-move verbs carry the orange tick; status glyphs carry their state color. Everything else is navy.

03In usespec §2 · icon beside word, never instead of

The product is text-forward: the icon is a recognition aid beside the word, not a replacement for it. Icon-alone is the rare exception (Ctrl+K, close, settings), and where it stands alone it carries an accessible name.

What this is testing

  • It doesn't look like anything else. Flat terminals, sharp joins, a heavier keyline — a stranger can't place it as Feather or Heroicons.
  • Every glyph means something. A sacred noun, a verb, or a real system/status meaning — nothing decorates, nothing fills a slot.
  • Color is rationed. Navy nearly everywhere; the blue tick only on Signal, the orange tick only on the dominant-move verbs, state colors only on status.
  • The family holds. Nouns and verbs read as one set — same grid, same weight, same terminals, the same edge-rule the cards use.
  • Icons support words. Beside the label, not instead of it; icon-alone is the rare exception and always named for a screen reader.
