A measured set of primitives

The composed cut treats every primitive as a tailored object — soft elevated surfaces, generous padding, rounded corners, gentle shadows. Public Sans does most of the reading work; DM Serif Display lifts the moments that earn weight; JetBrains Mono recedes almost entirely.

Color palette inherits canon Part II §1 (bright field) and §3 (orange for the dominant move, blue for system intelligence, green for healthy, amber for caution, red for risk). Typography inherits §2. The other variants take the same primitives in distinctly different voices.
