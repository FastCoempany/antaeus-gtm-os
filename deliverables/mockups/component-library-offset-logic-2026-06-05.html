The idea

Offset Logic. Controlled asymmetry is the signature. Eyebrows offset from cards. Action buttons partway outside card edges. Nav active state as a small offset block instead of a pill. Drawers step inward as the operator drills deeper. Lists allow a single row to break rank when something is genuinely elevated. Distinctiveness through spatial behavior — never decoration.

Canon preserved. Bright field. DM Serif Display for titles and modal headers. Public Sans for body. JetBrains Mono for eyebrows, codes, and timestamps. Orange for the dominant move; all canon color roles intact. The offsets are subtle by design — easy to overdo, so the discipline is restraint.
